# SignalAtlas

SignalAtlas aggregates research updates from leading XR and marine robotics labs
and presents them in a ranked dashboard.

## Features
- Automated research feed ingestion
- Lab-based research tracking
- Ranking by novelty and relevance
- Admin monitoring dashboard

## Stack
Next.js
Prisma
PostgreSQL
Vercel

## Local setup
1. npm install
2. set DATABASE_URL
3. prisma migrate dev
4. npm run dev
